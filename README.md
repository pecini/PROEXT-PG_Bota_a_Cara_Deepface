# bota_a_cara
Este é um dos projetos que compõem a instalação interativa “Bota a cara: vendo pelos olhos do algoritmo”,  do projeto “Dados e ativação comunitária sobre o uso de reconhecimento facial na segurança pública”, apoiado pela CAPES por meio do edital PROEXT-PG.  O objetivo é mostrar como funcionam os sistemas de RF e REF.
